# SmartHome API Documentation

Standalone Swagger UI for SmartHome Backend API.

## Local Development

```bash
npm install
npm start
```

Open http://localhost:3002 to view the API documentation.

## Deployment

Deploy to Vercel:
```bash
git init
git add .
git commit -m "Initial commit"
# Create a new repo on GitHub and push
git remote add origin <your-repo-url>
git push -u origin master
```

Then import the repo in Vercel dashboard.

## API Server

- **Production:** https://smart-home-backend-silk.vercel.app
- **Local:** http://localhost:3001
